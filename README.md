# Деревенская лавка

Сайт для продажи домашней деревенской продукции.

## Возможности

- Каталог продуктов и корзина
- Заявка на заказ с оплатой при получении (наличные / терминал)
- Кабинет владельца: товары, цены, тексты сайта, заявки

## Запуск

```bash
npm install
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000).

Кабинет: [http://localhost:3000/admin](http://localhost:3000/admin)

По умолчанию:
- логин: `admin`
- пароль: `lavka2026`

Смените данные в файле `.env.local` (см. `.env.example`).

## Данные

- Товары: `data/products.json`
- Тексты сайта: `data/settings.json`
- Заявки: `data/orders.json`
